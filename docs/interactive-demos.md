# Interactive Demo Block

Use this Markdoc tag in blog content:

```md
{% interactiveDemo preset="sorting" /%}
```

Supported `preset` values:

- `sorting`
- `traversal`
- `shortest-path`
- `chart`

Built-in controls:

- `Play`
- `Pause`
- `Step`
- `Reset`
- `Randomize`

No-JS fallback is included automatically via `noscript` content.
